# Reading List Tracker

A personal 20-week intellectual reading curriculum tracker with session logging, progress visualization, and gamification.

**Live Demo:** https://reading-list-orpin.vercel.app

## Features
- 20-week curriculum across 5 domains (Systems/Tech, Human/EQ, Creative/Craft, Society/Philosophy, Sci-Fi)
- Session timer with Pomodoro-style focus blocks
- GitHub-style contribution heatmap
- Command palette (Cmd+K) for quick navigation
- Streak tracking and badge system

## Tech Stack
React, Vite, Tailwind CSS, Framer Motion
